Privacy Policy

Last updated: April 6, 2026

Overview

ScoreBlur is a browser extension that automatically blurs sports scores to help you avoid spoilers. Your privacy is important to us, and this policy explains how we handle your data.

Data Collection

ScoreBlur does not automatically collect, store, or transmit any personal data.

We do not track your browsing activity. All score detection and blurring happens locally in your browser. No page content is ever sent to external servers.

The only exception is the optional feedback form described below, which you must actively choose to use.

Feedback Form

ScoreBlur includes an optional feedback form in the Settings page. When you choose to submit feedback, the following data is sent to Google Forms:

  • Feedback type (Bug, Feature Request, or Other)
  • Website URL where the issue occurred (if you provide it)
  • Your description of the issue or suggestion
  • Extension version number

This data is transmitted only when you explicitly click "Send Feedback." No data is sent automatically.

Payment Processing

ScoreBlur offers a 7-day free trial after installation, followed by a one-time payment to continue using the extension.

Payments are processed through Dodo Payments, a third-party service powered by Stripe. When you make a purchase:

  • You are redirected to a secure Stripe checkout page hosted on dodopayments.com
  • Your payment details (credit card, billing info) are handled entirely by Stripe — ScoreBlur never sees or stores this information
  • Dodo Payments returns only your payment status (paid or not paid) to the extension

ScoreBlur stores a minimal license record locally in your browser containing: payment status, trial end date, install date, and the timestamp of the last license check. This data never leaves your browser except during periodic license verification requests to Dodo Payments servers.

Local Storage

ScoreBlur uses Chrome's local storage API to save your preferences and license information, such as:

  • Whether the extension is enabled or disabled
  • Temporary pause settings (duration and type)
  • License status (paid/trial, trial end date, install date, last verification timestamp)

This data never leaves your browser and is not accessible to any third party, except for periodic license verification with Dodo Payments as described above.

Development Logging

Development builds of ScoreBlur include diagnostic logging stored temporarily in your browser's session storage. These logs are never transmitted to any server and are automatically cleared when your browser session ends. Production builds (the version you install from the Chrome Web Store) do not include this logging.

Permissions Explained

  • activeTab

    Allows ScoreBlur to access the content of the currently active tab to detect and blur scores. This permission is limited to the active tab only.

  • storage

    Used to save your extension preferences locally in your browser.

  • alarms

    Used to manage timed pause functionality (e.g., pause for 1 hour) and periodic license verification.

  • sidePanel

    Provides an optional side panel interface for quick access to ScoreBlur controls without leaving your current page.

  • Host permissions (all URLs)

    Required to detect and blur scores on any sports website you visit. ScoreBlur only processes page content to find score patterns — it does not read, store, or transmit any other page data.

Third-Party Services

  • Google Forms — Used to receive optional user feedback. No data is sent unless you actively submit the feedback form.
  • Dodo Payments / Stripe — Used for payment processing and license verification. Payment details are handled entirely by Stripe; ScoreBlur only receives your payment status.

ScoreBlur does not use any analytics platforms, advertising networks, or tracking services.

Changes to This Policy

If we update this privacy policy, the changes will be reflected on this page with an updated date. Significant changes will be communicated through the extension update notes.
